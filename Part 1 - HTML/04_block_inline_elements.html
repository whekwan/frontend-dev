Block elements

Block element width equals to the Device's width.

Block element always starts on a new line

block element contains top and bottom margin

Example: <h1>,<form>,<p>,<table> etc.


Inline elements

Inline element only takes as much width as necessary.

Inline element will not start on a new line

Inline element don't have top and bottom margin by default.

Example: <a>,<br>,<img>,<span> etc.
